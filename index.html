Política de Privacidade

Este aplicativo usa a API do WhatsApp para envio automatizado de mensagens com consentimento do usuário.

Não coletamos nem armazenamos dados pessoais sensíveis. Toda informação compartilhada é usada exclusivamente para responder aos usuários no WhatsApp.

Ao interagir com a assistente Dany, você concorda com esses termos.
